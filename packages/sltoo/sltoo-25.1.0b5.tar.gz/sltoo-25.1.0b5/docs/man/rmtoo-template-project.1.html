NAME

rmtoo-template-project - requirements management tool: template project

DESCRIPTION

rmtoo comes with a complet and ready to use template project.

The template project comes with only a few requirements and topics. The environment is set up in the way, that ’make’ can be used to create mostly all available artifacts.

To start with the template project, copy the directory doc/template_project and start editing.
