NAME

rmtoo-art-tlp1 - requirements management tool: tulip version 1 output

DESCRIPTION

This output module privides an interface to the graph visualization software tulip. With tulip it is possible to vizualize the requirements dependencies using different graph layouts.

Note that tulip works with graphs and not with digraphs. Therefore the direction of the dependency cannot be visualized.
