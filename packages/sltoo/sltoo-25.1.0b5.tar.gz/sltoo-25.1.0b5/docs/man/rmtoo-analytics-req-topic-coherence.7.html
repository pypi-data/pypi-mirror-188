NAME

rmtoo-analytics-req-topic-coherence - requirements management tool: analytics module for detecting topic coherence inside one requirement

DESCRIPTION

This analytics module checks for the coherence of each requirement inside a topic.

IDEA

A topic as used in rmtoo is a collection of requirements with one topic. Therefore each requirement should have a better coherence inside the topic than to the outside.

The difference to rmtoo-analytics-topic-coherence.7 is, that here the heuristic is based on one requirement whereas there the heuristic is based on the whole topic.

CONFIGURATION

This module cannot be configured.
